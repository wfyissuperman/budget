## 🧳 差旅费预算模块

### ✅ 参数设置

- 交通费：¥2000（往返）
- 补助标准：¥180 / 天 / 人
- 住宿标准（以北京为例）：¥500 / 晚
- 默认预算期望值：¥53,120
    
### 🚀 功能描述

该模块通过穷举搜索差旅人数、天数、次数组合，最小化总费用与期望预算之间的差值。

### 📤 输出示例

```
预算计算值与设定值的偏差为: 40.0  
最优解: 出差人数 = 4, 出差天数 = 6, 出差次数 = 2  
总费用: 53280.0  
人员交通费: 24000  
人员住宿费: 10000  
人员补助费: 19280  
```
